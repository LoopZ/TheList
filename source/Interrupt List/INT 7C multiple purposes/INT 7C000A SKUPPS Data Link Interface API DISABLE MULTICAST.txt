INT 7C - SK-UPPS Data Link Interface API - DISABLE MULTICAST
	AX = 000Ah
	BX = protocol ID (see #03849)
	ES:DI -> multicast address (6 bytes)
Return: AX = completion code (see #03846)
SeeAlso: AX=0009h,AX=0014h
