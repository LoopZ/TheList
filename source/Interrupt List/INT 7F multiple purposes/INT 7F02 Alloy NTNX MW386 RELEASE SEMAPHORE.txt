INT 7F - Alloy NTNX, MW386 - RELEASE SEMAPHORE
	AH = 02h
	DS:DX -> ASCIZ semaphore name (max 64 bytes)
Return: AL = status (see #03871)
	AH = semaphore owner if status=02h
SeeAlso: AH=00h,AH=01h,AH=42h
