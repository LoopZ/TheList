INT 10 - HUNTER 16 - SET BACKLIGHT TIMEOUT
	AX = 6501h
	BX = timeout in seconds (0-300)
Return: nothing
Desc:	set the idle time after which the backlight will turn off
SeeAlso: AH=64h,AX=6500h
