INT 10 - Chips & Technologies Extended BIOS - SET FLAT-PANEL VIDEO POLARITY
	AX = 5F5Ah
	BL = new video polarity
	    00h normal
	    01h inverted
Return: AL = 5Fh if function supported
	    AH = status
		00h unsuccessful
		01h successful
SeeAlso: AX=5F50h,AX=5F54h
