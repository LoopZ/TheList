INT 10 - Cirrus Logic BIOS v3.02 - LCD panel - SET CENTERING MODE
	AH = 12h
	BL = 90h
	AL = new position
	    00h centered
	    01h from top
	    02h from bottom
	    03h from top
Return: nothing
SeeAlso: AH=12h/BL=84h,AH=12h/BL=8Fh
