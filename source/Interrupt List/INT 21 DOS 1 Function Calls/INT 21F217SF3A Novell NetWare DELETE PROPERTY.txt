INT 21 - Novell NetWare - DELETE PROPERTY
	AX = F217h subfn 3Ah
	CX = length of request packet in bytes
	DX = 0000h (no reply packet)
	DS:SI -> request packet	(see #01936 at AH=E3h/SF=3Ah)
	ES:DI ignored
