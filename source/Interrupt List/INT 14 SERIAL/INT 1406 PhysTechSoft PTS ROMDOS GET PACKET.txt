INT 14 - PhysTechSoft PTS ROM-DOS - GET PACKET
	AH = 06h
	CX = buffer size for packet
	DX = port number
	DS:SI -> buffer for packet
Return: AH = error code (00h = no error, FFh = error)
	CX = packet size
SeeAlso: AH=05h"PTS ROM-DOS",INT 15/AX=E908h,INT 15/AX=E909h
