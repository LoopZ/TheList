CMOS 11h - IBM PS/2 - FIRST FIXED DISK DRIVE TYPE BYTE (00-FFh)
Note:	if IBM ESDI or SCSI drive controller is used, CMOS drive type will be
	  zero (00 - no drive) and INT 13h will be directed to controller ROM.
