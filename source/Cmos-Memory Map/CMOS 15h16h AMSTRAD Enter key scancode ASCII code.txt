CMOS 15h-16h - AMSTRAD - Enter key scancode/ASCII code
Size:	WORD
Desc:	specify the BIOS keycode for keyboard scancode 74h
Note:	default: 1C0Dh	- emulates Return key
SeeAlso: INT 09,CMOS 17h"AMSTRAD"
