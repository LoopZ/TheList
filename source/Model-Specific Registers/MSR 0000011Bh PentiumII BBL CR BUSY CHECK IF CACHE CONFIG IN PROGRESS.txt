MSR 0000011Bh - PentiumII - "BBL_CR_BUSY" CHECK IF CACHE CONFIG IN PROGRESS
Size:	1 bit
Access:	Read-Only
Note:	if bit 0 is set, an L2 cache configuration access command is in
	  progress
SeeAlso: MSR 00000088h,MSR 00000118h,MSR 0000011Ah
